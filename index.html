Construcción de Software

Creación de software funcional a través de actividades estructuradas

¿Qué es?

La construcción de software implica la creación de software significativo y funcional mediante una combinación de codificación, verificación, pruebas unitarias, pruebas de integración y depuración.

Relaciones

Este proceso está fuertemente vinculado con el proceso de diseño de software y el de pruebas de software, ya que incluye diseño y pruebas importantes del software.

Elementos de Configuración

La construcción de software generalmente produce la mayor cantidad de elementos de configuración que deben administrarse en un proyecto de software, como archivos de origen, documentación y casos de prueba.

La construcción de software es una de las etapas que lleva a la implementación dentro del ciclo de vida del desarrollo de software.
